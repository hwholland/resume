sap.m.ObjectMarker

The sap.m.ObjectMarker control represents one of the following predefined types:

  • Flagged
  • Favorite
  • Draft
  • Locked
  • LockedBy
  • Unsaved
  • UnsavedBy

Note: Use the LockedBy/UnsavedBy type along with the additionalInfo property to display the name of the user who locked/changed the object.
