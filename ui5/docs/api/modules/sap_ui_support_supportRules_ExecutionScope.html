sap.ui.support.ExecutionScope

Overview

ExecutionScope provides access to internal UI5 objects available for inspection

Usage

Each rule is passed three parameters when executed: check: oIssueManager, oCoreFacade, oScope An ExecutionScope instance is passed to every call of a rule's check method.
