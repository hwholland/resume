Class sap.ui.support.ExecutionScopeModule: sap/ui/support/supportRules/ExecutionScope

Overview

ExecutionScope provides access to internal UI5 objects available for inspection

Usage

Each rule is passed three parameters when executed: check: oIssueManager, oCoreFacade, oScope An ExecutionScope instance is passed to every call of a rule's check method. Available objects are collected depending on the settings passed to Support Assistant's entry point - the analyze method
