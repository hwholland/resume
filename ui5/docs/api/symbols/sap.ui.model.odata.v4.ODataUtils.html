sap.ui.model.odata.v4.ODataUtils.compare(vValue1, vValue2, vEdmType?): number
Compares the given OData values.
Parameters:
anyvValue1The first value to compare
anyvValue2The second value to compare
boolean|stringvEdmType?If true or "Decimal", the string values vValue1 and vValue2 are assumed to be valid "Edm.Decimal" or "Edm.Int64" values and are compared as a decimal number (only sign, integer and fraction digits; no exponential format). If "DateTime", the string values vValue1 and vValue2 are assumed to be valid "Edm.DateTimeOffset" values and are compared based on the corresponding number of milliseconds since 1 January, 1970 UTC. Otherwise the values are compared with the JavaScript operators === and >.
Returns:
numberThe result of the comparison: 0 if the values are equal, 1 if the first value is larger, -1 if the second value is larger, NaN if they cannot be compared

1.43.0
