Class sap.ui.core.format.NumberFormatModule: sap/ui/core/format/NumberFormat

The NumberFormat is a static class for formatting and parsing numeric values according to a set of format options.

stringoFormatOptions.style?defines the style of format. Valid values are 'short, 'long' or 'standard' (based on CLDR decimalFormat). This can be changed by setting either min/maxFractionDigits, decimals, shortDecimals or precision option.
sap.ui.core.format.NumberFormat.RoundingModeoFormatOptions.roundingMode?specifies a rounding behavior for discarding the digits after the maximum fraction digits defined by maxFractionDigits. Rounding will only be applied, if the passed value if of type number. This can be assigned by value in RoundingMode or a function which will be used for rounding the number. The function is called with two parameters: the number and how many decimal digits should be reserved.
booleanoFormatOptions.showMeasure?defines whether the measure according to the format is shown in the formatted string
booleanoFormatOptions.currencyCode?defines whether the currency is shown as code in currency format. The currency symbol is displayed when this is set to false and there's symbol defined for the given currency code.
stringoFormatOptions.currencyContext?It can be set either with 'standard' (the default value) or with 'accounting' for an accounting specific currency display
numberoFormatOptions.emptyString?, Default: NaN@since 1.30.0 defines what empty string is parsed as and what is formatted as empty string. The allowed values are only NaN, null or 0.
