Enum sap.ui.model.odata.v4.SubmitModeModule: sap/ui/model/odata/v4/SubmitMode

Modes to control the use of batch requests for a group ID.
Field Summary
sap.ui.model.odata.v4.SubmitMode.APIRequests associated with the group ID are sent in a batch request via sap.ui.model.odata.v4.ODataModel#submitBatch.
sap.ui.model.odata.v4.SubmitMode.AutoRequests associated with the group ID are sent in a batch request which is triggered automatically before rendering.
sap.ui.model.odata.v4.SubmitMode.DirectRequests associated with the group ID are sent directly without batch.
