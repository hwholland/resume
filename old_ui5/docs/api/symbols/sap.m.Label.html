Class sap.m.LabelModule: sap/m/Label

extends Control

Provides a textual label for other controls. Label appearance can be influenced by properties such as textAlign, design, displayOnly and wrapping. As of version 1.50 the default value of the wrapping property is set to false

Labels for required fields are marked with an asterisk.

Overview

Labels are used as titles for single controls or groups of controls.

Usage

When to use

  • It's recommended to use the Label in Form controls.
  • Use title case for labels.

When not to use

  • It is not recommended to use labels in Bold.

new sap.m.Label(sId?, mSettings?)

Constructor for a new Label.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • design : sap.m.LabelDesign (default: Standard)
    • text : string
    • textAlign : sap.ui.core.TextAlign (default: Begin)
    • textDirection : sap.ui.core.TextDirection (default: Inherit)
    • width : sap.ui.core.CSSSize (default: )
    • required : boolean (default: false)
    • displayOnly : boolean (default: false)
    • wrapping : boolean (default: false)
    • vAlign : sap.ui.core.VerticalAlign (default: Inherit)

  • Associations

    • labelFor : (sap.ui.core.ID | sap.ui.core.Control)

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
stringsId?ID for the new control, generated automatically if no ID is given
objectmSettings?Initial settings for the new control
