Module sap/m/NavContainer

Contained Namespaces, Types, Enums & Classes
sap.m.NavContainer The NavContainer control handles hierarchic navigation between Pages or other fullscreen controls.
sap.m.NavContainerChild sap.m.NavContainerChild is an artificial interface with the only purpose to bear the documentation of pseudo events triggered by sap.m.NavContainer on its child controls when navigation occurs and child controls are displayed/hidden.
