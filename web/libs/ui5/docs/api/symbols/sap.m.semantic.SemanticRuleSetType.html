Enum sap.m.semantic.SemanticRuleSetTypeModule: sap/m/library

Declares the type of semantic ruleset that will govern the styling and positioning of semantic content.
Since: 1.44.
Field Summary
sap.m.semantic.SemanticRuleSetType.ClassicThe default ruleset type, for which the Share Menu is always in the footer of the page.
sap.m.semantic.SemanticRuleSetType.OptimizedOffers an optimized user experience, with displaying the Share Menu in the header, rather than the footer, for Fullscreen mode.
