Class sap.m.NotificationListGroupModule: sap/m/NotificationListGroup

extends NotificationListBase

The NotificationListItemGroup control is used for grouping notification items of the same type.

Behavior

The group handles specific behavior for different usecases:

  • autoPriority - sets the group priority to the highest priority of an item in the group.
  • enableCollapseButtonWhenEmpty - displays a collapse button for an empty group.
  • showEmptyGroup - determines if the header/footer of an empty group is displayed.

new sap.m.NotificationListGroup(sId?, mSettings?)

Constructor for a new NotificationListGroup.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • collapsed : boolean (default: false)
    • autoPriority : boolean (default: true)
    • showEmptyGroup : boolean (default: false)
    • enableCollapseButtonWhenEmpty : boolean (default: false)

  • Aggregations

    • items : sap.m.NotificationListItem[] (default)

  • Events

    • onCollapse : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.m.NotificationListBase can be used as well.
