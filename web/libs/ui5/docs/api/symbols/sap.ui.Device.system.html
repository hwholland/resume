Namespace sap.ui.Device.systemModule: sap/ui/Device

Provides a basic categorization of the used device based on various indicators.

These indicators are for example the support of touch events, the screen size, the used operation system or the user agent of the browser.

Note: Depending on the capabilities of the device it is also possible that multiple flags are set to true.

Field Summary
sap.ui.Device.system.combiIf this flag is set to true, the device is recognized as a combination of a desktop system and tablet.
sap.ui.Device.system.desktopIf this flag is set to true, the device is recognized as a desktop system.
sap.ui.Device.system.phoneIf this flag is set to true, the device is recognized as a phone.
sap.ui.Device.system.tabletIf this flag is set to true, the device is recognized as a tablet.
Field Detail
sap.ui.Device.system.combi : boolean

If this flag is set to true, the device is recognized as a combination of a desktop system and tablet.

Furthermore, a CSS class sap-combi is added to the document root element.

Note: This property is mainly for Microsoft Windows 8 (and following) devices where the mouse and touch event may be supported natively by the browser being used. This property is set to true only when both mouse and touch event are natively supported.

sap.ui.Device.system.desktop : boolean

If this flag is set to true, the device is recognized as a desktop system.

Furthermore, a CSS class sap-desktop is added to the document root element.

sap.ui.Device.system.phone : boolean

If this flag is set to true, the device is recognized as a phone.

Furthermore, a CSS class sap-phone is added to the document root element.

sap.ui.Device.system.tablet : boolean

If this flag is set to true, the device is recognized as a tablet.

Furthermore, a CSS class sap-tablet is added to the document root element.

Note: This flag is also true for some browsers on desktop devices running on Windows 8 or higher. Also see the documentation for sap.ui.Device.system.combi devices. You can use the following logic to ensure that the current device is a tablet device:

if(sap.ui.Device.system.tablet && !sap.ui.Device.system.desktop){
	...tablet related commands...
}
