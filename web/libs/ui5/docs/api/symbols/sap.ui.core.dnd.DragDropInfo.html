new sap.ui.core.dnd.DragDropInfo(sId?, mSettings?)

Constructor for a new DragDropInfo.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • sourceAggregation : string
    • targetAggregation : string
    • dropEffect : sap.ui.core.dnd.DropEffect (default: Move)
    • dropPosition : sap.ui.core.dnd.DropPosition (default: On)
    • dropLayout : sap.ui.core.dnd.DropLayout (default: Vertical)

  • Associations

    • targetElement : (sap.ui.core.ID | sap.ui.core.Element)

  • Events

    • dragStart : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • dragEnter : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • drop : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.dnd.DragDropBase can be used as well.

Parameters:
stringsId?ID for the new DragDropInfo, generated automatically if no ID is given
objectmSettings?Initial settings for the DragDropInfo
