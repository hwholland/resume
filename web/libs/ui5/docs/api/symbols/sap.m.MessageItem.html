Class sap.m.MessageItemModule: sap/m/MessageItem

extends Item

A wrapper control used to hold different types of system messages.

Structure

The message item holds the basic set of properties for a system message:

  • Type, title, subtitle and description
  • If the description contains markup, the markupDescription needs to be set to true, to ensure it is interpreted correctly.
  • If the long text description can be specified by a URL by setting, the longtextUrl property.
  • The message item can have a single sap.m.Link after the description. It is stored in the link aggregation.

    • Usage

    Note: The MessageItem control replaces sap.m.MessagePopoverItem as a more generic wrapper for messages.


    Since: 1.46.

new sap.m.MessageItem(sId?, mSettings?)

Constructor for a new MessageItem.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • type : sap.ui.core.MessageType (default: Error)
    • title : string (default: )
    • subtitle : string
    • description : string (default: )
    • markupDescription : boolean (default: false)
    • longtextUrl : sap.ui.core.URI
    • counter : int
    • groupName : string (default: )

  • Aggregations

    • link : sap.m.Link (default)

In addition, all settings applicable to the base type sap.ui.core.Item can be used as well.
