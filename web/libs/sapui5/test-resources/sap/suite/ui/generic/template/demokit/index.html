Sample Application based on Smart Templates

This is a reference implementation of the ListReport/Object Page design pattern in Fiori.
Choose one of the access points below to launch the app.

Demokit/SDK

These files can be used to run the application from within the Demokit/SDK (using the UI5 version shipping with the SDK).
Run local
  • demokit.html - start the Manage Products technical app with mock data
  • demokit.html - start the Sales Order app with mock data
Run local with Runtime Adaptation
  • demokit.html - start the Manage Products technical app with mock data
  • demokit.html - start the Sales Order app with mock data
Run on WebIDE
  • demokitWebIDE.html - start the Manage Products technical app with mock data
  • demokitWebIDE.html - start the Sales Order app with mock data
Run on WebIDE with Runtime Adaptation
  • demokitWebIDE.html - start the Manage Products technical app with mock data
  • demokitWebIDE.html - start the Sales Order app with mock data
