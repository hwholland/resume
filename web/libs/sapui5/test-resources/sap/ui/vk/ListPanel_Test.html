Visual Interaction ListPanelStack Sample

  • Testing the ListPanelStack
  • A ListPanelStack aggregates multiple ListPanels
  • A ListPanels can be expanded and collapsed to just the header
  • If all ListPanels are collapsed the ListPanelStack collapses as well to a vertical list of buttons
  • Pressing a button will expand the ListPanelStack and the ListPanel the button belongs to
  • UI Element sizes vary dependent on size mode. Default is Cozy for mobile device. On Desktop you can enable Compact mode
