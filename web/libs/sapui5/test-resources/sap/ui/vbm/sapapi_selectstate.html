Visual Business UI5: Sample should demonstrate different kind of select state behavior of some visual objects.

  • Only one visual object in a column can be selected. ( Selection cardinality 1:1, initially no selection done )
  • The hot scale value is increasing from top to bottom.
  • The hot color in the first row does a transparency shift. The select color does a hue shift.
  • The hot color in the second row does a 90 degree color hue shift. The select color does a lightness increase by 50%.
  • The hot color in the third row is set to red. The select color is fixed to a blue color with transparency.
  • Important: the hot scale/select shifts are not available in all objects. ( Pies are not supported )
