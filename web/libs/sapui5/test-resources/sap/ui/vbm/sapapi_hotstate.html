Visual Business UI5: Sample should demonstrate different kind of hotstate behavior of some visual objects.

  • The hot scale value is increasing from top to bottom. ( except for geocircles )
  • The hot color in the first row does a transparency shift.
  • The hot color in the second row does a 90 degree color hue shift.
  • The hot color in the third row is set to red.
  • Important: the hot scale/shifts are not available in all objects.
