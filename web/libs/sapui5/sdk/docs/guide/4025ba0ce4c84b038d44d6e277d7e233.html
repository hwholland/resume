Running SAPUI5 Applications on SAP HANA

To run a SAPUI5 application on SAP HANA, you either expose the respective package via HTTP or you create the SAPUI5 application inside a SAP HANA XS application.

Context

For more information, see SAPUI5 for SAP HANA Development Tutorials in the documentation for SAP HANA on SAP Help Portal at http://help.sap.com/hana_platform/ under  Development Information  SAP HANA Developer Guide  Building UIs .

Procedure

  1. Ensure that you have changed the bootstrap tag in index.html to enable the application to access the SAPUI5 libraries on the xsengine as follows: src="resources/sap-ui-core.js" to src="/sap/ui5/1/resources/sap-ui-core.js".
  2. To expose the SAPUI5 application project:
    • If the package that contains your SAPUI5 application project has not been exposed via HTTP already, you need to create .xsapp and .xsaccess files.
    • This is not necessary, if the project was created as a sub-package of an SAP HANA XS application.
  3. Activate the project on the SAP HANA Web server.
  4. To start the UI application on the SAP HANA Web server, use the following URL: <xsengine protocol>://<xsengine server>:<xsengine port>/<SAPUI5 application name>/WebContent/index.html . Replace the placeholders accordingly.
