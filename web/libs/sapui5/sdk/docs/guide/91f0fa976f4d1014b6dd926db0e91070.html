Models and Data Binding in Applications

Models and data binding is used in SAPUI5 applications to connect and synchronize two data or information sources.

The following sections describe the process for data binding, how a model instance is created and assigned to the UI and how the controls on the UI are bound to the model. SAPUI5 provides several binding modes and provides multimodel support.
