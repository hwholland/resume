Configuration of Theme and Styling for MAKit Charts

The SAPUI5 theme parameters to allow dynamic styling and theming for MAKit charts.

You can adjust the following parameters in the theme editor:

Theme Parameter Description
sapUiValueBubbleHightlight Defines the color that is used to highlight the value bubble
sapUiValueBubbleLegendBG Defines the background color fo the value bubble legend
sapUiValueBubbleLegendTextColor Defines the text color for the value bubble legend
sapUiChartRangeSelectorFontSize Defines the font size used for the range selector
sapUiChartRangeSelectorFontFamily Defines the font family used for the range selector
sapUiChartRangeSelectorFontColor Defines the font size used for the range selector
sapUiChartTableFontSize Defines the font size used for the table view
sapUiChartTableFontFamily Defines the font family used for the table view
sapUiFillPaletteColor1 to sapUiFillPaletteColor16 Defines the color palette that can be used for drawing chart items, for example the bar
