Delete Personalized KPI Tiles

The Delete Personalized KPI Tiles app allows you to delete KPI tiles created using the Save as Tile feature of the generic drill-down app. As Delete Personalized KPI Tiles is not a modeler app, it is not available under modeler apps catalog. You can add this app to the Fiori Launchpad from the SAP: KPIs catalog.

Key Features

  • All the KPI tiles created using the Save as Tile feature of the generic drill-down app are listed.

  • You can delete the desired KPI tile.
