new sap.viz.ui5.types.legend.Common(sId?, mSettings?)
Constructor for a new ui5/types/legend/Common.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties
    • visible : boolean (default: true)
    • formatString : string (default: 'null')
    • isHierarchical : boolean (default: false)
    • isScrollable : boolean (default: false)
    • position : sap.viz.ui5.types.legend.Common_position (default: sap.viz.ui5.types.legend.Common_position.right)
    • type : sap.viz.ui5.types.legend.Common_type (default: sap.viz.ui5.types.legend.Common_type.ColorLegend)
    • alignment : sap.viz.ui5.types.legend.Common_alignment (default: sap.viz.ui5.types.legend.Common_alignment.start)
    • drawingEffect : sap.viz.ui5.types.legend.Common_drawingEffect (default: sap.viz.ui5.types.legend.Common_drawingEffect.normal)
  • Aggregations
    • title : sap.viz.ui5.types.legend.Common_title
  • Associations
    • Events

      In addition, all settings applicable to the base type sap.viz.ui5.core.BaseStructuredType can be used as well.
