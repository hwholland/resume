Class sap.ui.comp.smartfield.SmartField

extends Control

The SmartField Control is a wrapper for other controls. It interprets OData meta data to determine the control that has to be instantiated. The OData entity is deduced from the control's binding context. The OData entity's property that is changed or displayed with the control is deduced from the control's value property. The control's value property can also be bound to a property of a JSON model. In this case the SmartField uses its jsontype property to determine the control to be created.

new sap.ui.comp.smartfield.SmartField(sId?, mSettings?)
Constructor for a new smartfield/SmartField.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • value : any
    • enabled : boolean (default: true)
    • entitySet : string
    • editable : boolean (default: true)
    • width : sap.ui.core.CSSSize
    • textAlign : sap.ui.core.TextAlign
    • placeholder : string
    • name : string
    • valueState : sap.ui.core.ValueState
    • valueStateText : string
    • showValueStateMessage : boolean (default: true)
    • jsontype : sap.ui.comp.smartfield.JSONType
    • mandatory : boolean (default: false)
    • maxLength : int (default: 0)
    • showSuggestion : boolean (default: true)
    • showValueHelp : boolean (default: true)
    • showLabel : boolean (default: true)
    • textLabel : string
    • tooltipLabel : string
  • Aggregations
    • configuration : sap.ui.comp.smartfield.Configuration
  • Events
    • entitySetFound : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • change : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • initialise : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • visibleChanged : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
{string} sId? id for the new control, generated automatically if no id is given
{object} mSettings? initial settings for the new control
