new sap.suite.ui.commons.BulletChart(sId?, mSettings?)
Constructor for a new BulletChart.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties
    • size : sap.suite.ui.commons.InfoTileSize (default: sap.suite.ui.commons.InfoTileSize.Auto)
    • mode : sap.suite.ui.commons.BulletChartMode (default: sap.suite.ui.commons.BulletChartMode.Actual)
    • scale : string
    • forecastValue : float
    • targetValue : float
    • minValue : float
    • maxValue : float
    • showActualValue : boolean (default: true)
    • showDeltaValue : boolean (default: false)
    • showTargetValue : boolean (default: true)
    • showValueMarker : boolean (default: false)
    • actualValueLabel : string
    • deltaValueLabel : string
    • targetValueLabel : string
    • width : sap.ui.core.CSSSize
    • scaleColor : sap.suite.ui.commons.CommonBackground (default: sap.suite.ui.commons.CommonBackground.MediumLight)
  • Aggregations
    • actual : sap.suite.ui.commons.BulletChartData
    • thresholds : sap.suite.ui.commons.BulletChartData[]
  • Associations
    • Events
      • press : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    Parameters:
    {string} sId? id for the new control, generated automatically if no id is given
    {object} mSettings? initial settings for the new control
