Class sap.ui.comp.valuehelpdialog.ValueHelpDialog

extends Dialog

The ValueHelpDialog Control can be used to implement an F4 value help for a multi-input field.

new sap.ui.comp.valuehelpdialog.ValueHelpDialog(sId?, mSettings?)
Constructor for a new valuehelpdialog/ValueHelpDialog.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • basicSearchText : string
    • supportMultiselect : boolean (default: true)
    • supportRanges : boolean (default: false)
    • descriptionKey : string
    • key : string
    • supportRangesOnly : boolean (default: false)
    • maxIncludeRanges : string (default: -1)
    • maxExcludeRanges : string (default: -1)
    • displayFormat : string
    • tokenDisplayBehaviour : string
    • filterMode : boolean (default: false)
  • Aggregations
    • filterBar : sap.ui.core.Control
  • Events
    • ok : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • cancel : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.m.Dialog can be used as well.

Parameters:
{string} sId? id for the new control, generated automatically if no id is given
{object} mSettings? initial settings for the new control
