Class sap.ui.comp.smartform.SmartForm

extends Control

The SmartForm renders a form (sap.ui.layout.form.Form) and makes use of OData metadata annotations. If the controls to be displayed inside the form are bound to an OData model the label is taken from the metadata annotation (sap:label) if not specified in the XML view.

new sap.ui.comp.smartform.SmartForm(sId?, mSettings?)
Constructor for a new smartform/SmartForm.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • title : string
    • useHorizontalLayout : boolean
    • checkButton : boolean (default: false)
    • entityType : string
    • expandable : boolean (default: false)
    • expanded : boolean
    • editTogglable : boolean (default: false)
    • editable : boolean (default: false)
    • ignoredFields : string
  • Aggregations
    • groups : sap.ui.comp.smartform.Group[]
    • layout : sap.ui.comp.smartform.Layout
    • semanticObjectController : sap.ui.comp.navpopover.SemanticObjectController
    • customToolbar : sap.m.Toolbar
  • Events
    • editToggled : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • checked : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
{string} sId? id for the new control, generated automatically if no id is given
{object} mSettings? initial settings for the new control
