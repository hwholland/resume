Namespace sap.suite.ui.commons.ProcessFlowZoomLevel

The zoom level defines level of details for the node and how much space the process flow requires.

Field Summary
sap.suite.ui.commons.ProcessFlowZoomLevel.Four The full details with normal font size
sap.suite.ui.commons.ProcessFlowZoomLevel.One zoom level for least details - only icon is displayed
sap.suite.ui.commons.ProcessFlowZoomLevel.Three The full detail view of the node but with smaller font size
sap.suite.ui.commons.ProcessFlowZoomLevel.Two The detail is icon, title text and no additional texts
