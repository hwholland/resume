Namespace sap.viz.ui5.api.env.Format

Provides methods set global formatter for VizFrame.

As Format is a static class, a jQuery.sap.require("sap.viz.ui5.api.env.Format"); statement must be explicitly executed before the class can be used. Example: 

  jQuery.sap.require("sap.viz.ui5.api.env.Format");


Since: 1.24.0.
