Class sap.ui.comp.smarttable.SmartTable

extends VBox

The SmartTable control creates a table based on OData metadata and the configuration specified. The entitySet attribute must be specified to use the control. This attribute is used to fetch fields from OData metadata, from which columns will be generated.
Note that this attribute is not dynamic and cannot be changed once the control has been initialized! It can also be used to fetch the actual table data based on the tableType attribute. This control will render a standard, analytical, or responsive table.

Experimental API:Since version 1.25. The SmartTable will be productised soon.
