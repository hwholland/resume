new sap.ui.comp.personalization.BaseController(sId?, mSettings?)
The BaseController is a base class for personalization Controller like e.g. FilterController, SortController etc. *

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • type : string
    • model : sap.ui.model.json.JSONModel
    • persistentDataRestore : object
  • Events
    • beforePotentialTableChange : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • afterPotentialTableChange : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.base.ManagedObject can be used as well.

Parameters:
{string} sId? id for the new control, generated automatically if no id is given
{object} mSettings? initial settings for the new control
