new sap.ui.comp.smartmultiedit.Container(sId?, mSettings?)

Constructor for a new sap.ui.comp.smartmultiedit.Container.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • entitySet : string
    • contexts : sap.ui.model.Context[] (default: [])

  • Aggregations

    • layout : sap.ui.comp.smartform.SmartForm (default)

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
stringsId?ID for the new control, generated automatically if no ID is given
objectmSettings?Initial settings for the new control

getAllUpdatedContexts(merge?): Promise

Provides updated objects, including the values of all sap.ui.comp.smartmultiedit.Field instances in the layout aggregation.

The function returns a Promise that is resolved when each sap.ui.mode.Context in the contexts property is copied and updated. The fulfilled function is called with an array that contains an object for each context in the contexts property.
Each object in this array contains:

  • A context property that corresponds to the respective context.
  • A data property that contains the updated data object.
Parameters:
booleanmerge?If true, the returned data structure includes all entity set properties per context.
Returns:
PromiseA Promise that is resolved when all contexts (sap.ui.model.Context) are copied and updated.
