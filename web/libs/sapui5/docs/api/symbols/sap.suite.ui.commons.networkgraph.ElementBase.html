new sap.suite.ui.commons.networkgraph.ElementBase()

Constructor for a new ElementBase.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • title : string
    • description : string
    • status : string (default: sap.suite.ui.commons.networkgraph.ElementStatus.Standard)

  • Aggregations

    • attributes : sap.suite.ui.commons.networkgraph.ElementAttribute[]

In addition, all settings applicable to the base type sap.suite.ui.commons.networkgraph.SvgBase can be used as well.
