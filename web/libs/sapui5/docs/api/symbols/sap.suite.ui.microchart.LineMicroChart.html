Class sap.suite.ui.microchart.LineMicroChartModule: sap/suite/ui/microchart/LineMicroChart

extends Control
Chart that displays the history of values as segmented lines along a threshold line. The scale is optional and showing the points is also optional.
Since: 1.48.0.

new sap.suite.ui.microchart.LineMicroChart(sId?, mSettings?)

Constructor for a new LineMicroChart control.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • size : sap.m.Size (default: Auto)
    • threshold : float (default: 0)
    • minXValue : float
    • maxXValue : float
    • minYValue : float
    • maxYValue : float
    • leftTopLabel : string
    • rightTopLabel : string
    • leftBottomLabel : string
    • rightBottomLabel : string
    • color : any (default: Neutral)
    • showPoints : boolean (default: false)

  • Aggregations

    • points : sap.suite.ui.microchart.LineMicroChartPoint[] (default)

  • Events

    • press : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
stringsId?ID for the new control, generated automatically if no ID is given
objectmSettings?Initial settings for the new control

getColor(): any

Gets current value of property color.

Describes the color of the chart. In conjunction with emphasized points, it is only used if all points have the sap.m.ValueColor.Neutral color. The color can be set as an sap.m.ValueCSSColor or as a plain object. It has the 'above' and 'below' properties that determine the color of the graph above and below the threshold, respectively.

Default value is Neutral.

anyValue of property color

getThreshold(): float

Gets current value of property threshold.

Determines the chart threshold which is used for vertical normalization. If the threshold does not belong to the value range given by minYValue...maxYValue, the threshold is ignored. By setting the threshold property's value to null, the threshold is disabled and excluded from range calculations.

Default value is 0.
