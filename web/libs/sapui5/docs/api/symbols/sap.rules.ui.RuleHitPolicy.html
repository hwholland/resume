Enum sap.rules.ui.RuleHitPolicyModule: sap/rules/ui/library

An enumeration that defines the output when more than one rule in the decision table is matched for a given set of inputs.
Field Summary
sap.rules.ui.RuleHitPolicy.AllMatchSpecifies that all conditions that match the input (each matching row in the decision table) are returned as an output.
sap.rules.ui.RuleHitPolicy.FirstMatchSpecifies that only the first condition that matches the input (the first matching row by order in the decision table) is returned as an output.
