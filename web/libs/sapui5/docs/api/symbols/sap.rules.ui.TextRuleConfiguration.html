Constructor Detail
new sap.rules.ui.TextRuleConfiguration(sId?, mSettings?)

Constructor for a new TextRuleConfiguration.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • enableSettings : boolean (default: false)
    • enableElse : boolean (default: true)
    • enableElseIf : boolean (default: true)

  • Events

    • change : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Element can be used as well.

Parameters:
stringsId?id for the new control, generated automatically if no id is given
objectmSettings?initial settings for the new control
