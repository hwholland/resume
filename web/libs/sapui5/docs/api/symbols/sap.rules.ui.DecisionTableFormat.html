Enum sap.rules.ui.DecisionTableFormatModule: sap/rules/ui/library

An enumeration that decides the rendering format for decisionTable.
Field Summary
sap.rules.ui.DecisionTableFormat.RuleFormatSpecifies that the rendering is at the rule level.
sap.rules.ui.DecisionTableFormat.CellFormatSpecifies that the rendering of a cell is based on cellFormat and each cell can have a different format during rendering
Field Detail
sap.rules.ui.DecisionTableFormat.RuleFormat : string
Specifies that the rendering is at the rule level. The value set here is applicable for all the cells in the rule.
Default Value:
sap.rules.ui.DecisionTableFormat.CellFormat : string
Specifies that the rendering of a cell is based on cellFormat and each cell can have a different format during rendering
