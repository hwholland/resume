new sap.ui.vk.RedlineElementEllipse()

Redline element control for ellipse.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • radiusX : float (default: 0.0001)
    • radiusY : float (default: 0.0001)
    • fillColor : sap.ui.core.CSSColor (default: rgba(0, 0, 0, 0))

In addition, all settings applicable to the base type sap.ui.vk.RedlineElement can be used as well.
