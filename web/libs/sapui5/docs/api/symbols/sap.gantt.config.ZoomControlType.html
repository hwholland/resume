Field Detail
sap.gantt.config.ZoomControlType.ButtonsOnly
Uses the ButtonsOnly control to modify the time zoom rate. The SliderOnly control only consists of zoom in and zoom out buttons.
sap.gantt.config.ZoomControlType.None
Display no zoom control
sap.gantt.config.ZoomControlType.Select
Uses the Select control to modify the time zoom rate. The Select control consists of a drop down list to select a suitable zoom rate.
sap.gantt.config.ZoomControlType.SliderOnly
Uses the SliderOnly control to modify the time zoom rate. The SliderOnly control only consists of a slider.
sap.gantt.config.ZoomControlType.SliderWithButtons
Uses the SliderWithButtons control to modify the time zoom rate. The SliderWithButtons control consists of zoom in and zoom out magnifier buttons and a slider.
