Enum sap.f.AvatarImageFitTypeModule: sap/f/library

Types of image size and position that determine how an image fits in the sap.f.Avatar control area.
Since: 1.46.
Field Summary

sap.f.AvatarImageFitType.CoverThe image is scaled to be large enough so that the control area is completely covered.
