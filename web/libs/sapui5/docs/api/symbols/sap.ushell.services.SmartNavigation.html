new sap.ushell.services.SmartNavigation()

Constructs an instance of SmartNavigation.

The constructed service provides an enhancement on CrossApplicationNavigation#getLinks and CrossApplicationNavigation#toExternal. In order for an application to leverage this enhancement, it is pertinent that the application uses SmartNavigation#toExternal for naviagtion. Hence the caller can subsequently use SmartNavigation#getLinks with the outcome that it sorts the resulting list in the order of frequency of Attempted navigation from the application to respective links.

Attempted in the previous paragraph is emphasized due to the fact that a click on the link will cause an increment of the frequency count, regardless of wether the navigation was successful or not.

Note that an instance of this service should be obtained with sap.ushell.services.Container#getService( "SmartNavigation" )

Method Detail
getLinks(): jQuery.Deferred.promise

Resolves the given semantic object (or action) and business parameters to a list of links available to the user, sorted according their relevance to the calling application.

The relevance of link is defined by the frequency with which a navigation activity from the calling application to that link occurs.

Internally, this method delegates to sap.ushell.services.CrossApplicationNavigation#getLinks and then sorts the resulting list accordingly.

Returns:
jQuery.Deferred.promiseA promise that resolves with an array of link objects sorted according to their relevance to the calling application.

trackNavigation(oArguments): object
Tracks a navigation to a valid intent if provided via arguments but does not perform the navigation itself. If no valid intent was provided tracking will be prevented. The intent has to consist of SemanticObject and Action. It may be passed as complete shellHash (presidence) or as individual parts Additional parameters will not be part of the tracking and ignored This Method can be used to track a click if the actual navigation was triggered via clicking a link on the UI.
Parameters:
objectoArguments

The navigation target as object, for example:

{ target: { shellHash: 'SaleOrder-display' } }

or

{ target: { semanticObject: 'SalesOrder', action: 'action' } }

Returns:
object

promise the new item created for tracking
