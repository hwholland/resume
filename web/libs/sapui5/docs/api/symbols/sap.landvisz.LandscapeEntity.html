new sap.landvisz.LandscapeEntity(sId?, mSettings?)

Constructor for a new LandscapeEntity.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

    • systemName : string
    • type : sap.landvisz.LandscapeObject
    • qualifierText : string
    • qualifierTooltip : string
    • qualifierType : string
    • renderingSize : sap.landvisz.EntityCSSSize (default: sap.landvisz.EntityCSSSize.Regular)
    • defaultState : string
    • description : string
    • actions : object
    • systemStatus : sap.landvisz.ModelingStatus (default: sap.landvisz.ModelingStatus.NORMAL)
    • statusTooltip : string
    • explodeViewWidth : sap.ui.core.CSSSize
    • explodeViewHeight : sap.ui.core.CSSSize
    • showCustomActions : boolean (default: true)
    • showEntityActions : boolean (default: true)
    • systemId : string
    • stateIconSrc : string
    • stateIconTooltip : string
    • componentType : sap.landvisz.ComponentType
    • componentTypeTooltip : string
    • trackSystemCount : string

  • Aggregations

    • dataContainers : sap.landvisz.internal.DataContainer[]
    • actionBar : sap.landvisz.internal.ActionBar[]
    • entityStatus : sap.ui.commons.Image

  • Associations

    • Events

      • mouseOverIdenIcon : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
      • statusSelect : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
