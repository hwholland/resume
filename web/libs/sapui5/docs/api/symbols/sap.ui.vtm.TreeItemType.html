Enum sap.ui.vtm.TreeItemTypeModule: sap/ui/vtm/TreeItemType

Enumeration specifying the type(s) of tree item to match during tree traversal.
Experimental API:
Since 1.50.0 This type is experimental and might be modified or removed in future versions.
Field Summary
sap.ui.vtm.TreeItemType.ExcludedA tree item that is in the object hierarchy of the tree model, but cannot be shown in the tree.
sap.ui.vtm.TreeItemType.IncludedA tree item that can be shown in the tree.

Field Detail
sap.ui.vtm.TreeItemType.Excluded : string
A tree item that is in the object hierarchy of the tree model, but cannot be shown in the tree. A tree item is an excluded tree item when any ancestor of the tree item is included in the excludedChildren collection of its parent.
Default Value:
sap.ui.vtm.TreeItemType.Included : string
A tree item that can be shown in the tree. A tree item is an included tree item when it is a root tree item or is included in the includedChildren collection of its parent.
Default Value:
sap.ui.vtm.TreeItemType.IncludedOrExcluded : string
Matches either included or excluded tree items.
Default Value:
