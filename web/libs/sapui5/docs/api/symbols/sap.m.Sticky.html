Enum sap.m.StickyModule: sap/m/library

Defines which area of the control remains fixed at the top of the page during vertical scrolling as long as the control is in the viewport.
Since: 1.54.
Field Summary
sap.m.Sticky.ColumnHeadersOnly column headers remain in a fixed position.
sap.m.Sticky.NoneNo area remains in a fixed position.
Field Detail
sap.m.Sticky.ColumnHeaders : string
Only column headers remain in a fixed position. If no column headers are available, this option behaves the same as None.
Default Value:
sap.m.Sticky.None : string
No area remains in a fixed position.
