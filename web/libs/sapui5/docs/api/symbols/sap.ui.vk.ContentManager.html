Abstract Class sap.ui.vk.ContentManagerModule: sap/ui/vk/ContentManager

extends ManagedObject
known direct subclasses: ContentManager, ContentManager, ImageContentManager

Provides a base loader interface.

To load content a concrete loader class is to be used.

Experimental API:
Since 1.50.0 This class is experimental and might be modified or removed in future versions.

new sap.ui.vk.ContentManager(sId?, mSettings?)

Constructor for a new ContentManager.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Events

    • contentChangesStarted : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • contentChangesFinished : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • contentChangesProgress : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.base.ManagedObject can be used as well.

Parameters:
stringsId?ID for the new ContentManager object. Generated automatically if no ID is given.
objectmSettings?Initial settings for the new ContentConnector object.
