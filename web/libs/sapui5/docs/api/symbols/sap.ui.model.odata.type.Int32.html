Class sap.ui.model.odata.type.Int32Module: sap/ui/model/odata/type/Int32

extends Int

This class represents the OData primitive type Edm.Int32.

In both sap.ui.model.odata.v2.ODataModel and sap.ui.model.odata.v4.ODataModel this type is represented as a number.
