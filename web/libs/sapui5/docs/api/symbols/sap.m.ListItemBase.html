Class sap.m.ListItemBaseModule: sap/m/ListItemBase

extends Control
known direct subclasses: ActionListItem, BarListItem, ClusterListItem, ColumnListItem, CustomListItem, DisplayListItem, FacetFilterItem, FeedItemHeader, FeedListItem, GroupHeaderListItem, InputListItem, NotificationListBase, ObjectListItem, StandardListItem, TreeItemBase
ListItemBase contains the base features of all specific list items. Note: If not mentioned otherwise in the individual subclasses, list items must only be used in the items aggregation of sap.m.ListBase controls.

Constructor Detail
new sap.m.ListItemBase(sId?, mSettings?)

Constructor for a new ListItemBase.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • type : sap.m.ListType (default: Inactive)
    • visible : boolean (default: true)
    • unread : boolean (default: false)
    • selected : boolean (default: false)
    • counter : int
    • highlight : sap.ui.core.MessageType (default: None)

  • Associations

    • ariaLabelledBy : (sap.ui.core.ID | sap.ui.core.Control)[]

  • Events

    • tap : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • detailTap : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • press : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • detailPress : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
stringsId?Id for the new control, generated automatically if no id is given
objectmSettings?Initial settings for the new control
