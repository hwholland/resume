Class sap.ui.vk.NodeProxyModule: sap/ui/vk/NodeProxy

extends ManagedObject
known direct subclasses: NodeProxy, NodeProxy

Provides a proxy object to the node in the node hierarchy.

Objects of this type should only be created with the sap.ui.vk.NodeHierarchy.createNodeProxy method. and destroyed with the sap.ui.vk.NodeHierarchy.destroyNodeProxy method.

Experimental API:
Since 1.32.0 This class is experimental and might be modified or removed in future versions.

Constructor Detail
new sap.ui.vk.NodeProxy()

Constructor for a new NodeProxy.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • nodeRef : any
    • veIds : object[]
    • name : string
    • localMatrix : sap.ui.vk.TransformationMatrix
    • worldMatrix : sap.ui.vk.TransformationMatrix
    • opacity : float
    • tintColorABGR : int
    • tintColor : sap.ui.core.CSSColor
    • nodeMetadata : object
    • hasChildren : boolean
    • closed : boolean

In addition, all settings applicable to the base type sap.ui.base.ManagedObject can be used as well.
