Enabling Variant Management

Variants allow you to store settings that users create for the smart filter bar, such as selection fields and layout, and for the smart table, such as sorting and visible columns.

If you want, you can enable harmonized variant management for the list report. In this case, any variants created by the user for the smart filter bar and smart table are saved together using the variant for the smart filter bar. Code Sample

In the manifest.json file, the smartVariantManagement variable is set to false by default. To enable smart variant management, set it to true as shown below:

"sap.ui.generic.app": {

        "_version":"1.1.0",

        "pages": [

            {

                "entitySet": "XXXXXX_Product",

                "component": {

                    "name": "sap.suite.ui.generic.template.ListReport",

                    "list": true,

                    "settings" : {

                        "gridTable" : false,

                        "multiSelect": false,

                        "smartVariantManagement": true
