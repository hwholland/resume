Extending Generated Apps

You can extend your application if needed.

The extension concept for SAP Fiori elements follows the SAPUI5 extension concept. For more information, see Extending Apps.
Caution Extensions should be used with caution and only if you cannot produce the required behavior by other means, such as manifest settings or annotations.

For more information, see also API for Developers of Template Extensions.
