Refresh API

Use the refreshBinding function (in Component.js) to refresh the elements of header, filter, and content area.

This helps you to reload the data for all components configured in ALP without triggering the browser refresh. The selections in the filter bar (compact or visual filter) and the personalization (Sorting, Grouping and so on) settings are retained even after the refresh.
Note The main chart selection is lost during refresh. This may result in the change of table data when AutoHide is ON for filter mode.
Sample Code Refresh through the controller extensions defined by the application
//By calling refreshBinding function directly .

       var component = this.getOwnerComponent();

       component.refreshBinding();
