Managing Variants

Allows you to manage variants with different structures in the filter and content areas.

Context

ALP provides page-level and control-level variant management.

Use page-level variant management to capture filter selection, filter mode, view mode, auto hide icon state (eye-icon), chart and table configuration (measures and dimensions), sort order, and grouping. By default, the page-level variant is enabled.

Use control-level variant management to define separate variants for filter, chart, and table sections. Set smartVariantManagement=false in the app-descriptor file to use control-level variant managment. The filter variant stores filter area selections and also the filter mode. When you load the stored variant, both the filter area selection and filter mode appear. You cannot:
  • Turn off variant management for selected controls (filter, chart, or table area)
  • Use control-level variant management and page-level variant management together
Note

The SAPUI5 standard class sap.ui.comp.variants.VariantManagement supports and manages variants.
