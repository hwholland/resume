Maintaining Standard Text for Smart Tables

In a smart table, if the system does not find any entries when using the smart filter, a standard UI text that you can adapt in the i18n file of your app is displayed.

If you want to display your own text instead of the standard text, use the NOITEMS_SMARTTABLE key.
Note This text is not in the i18n file by default. You have to insert it if you want to change the standard text.
