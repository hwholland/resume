Adapting the Smart Filter Bar

For the SmartFilterBar, you can define application-specific selection fields by using com.sap.vocabularies.UI.v1.SelectionFields and field groups for the filter popup.

Note that the field groups are currently used in the SmartFilter only to group the fields. The grouping can only be seen in the Adapt Filters popup. Any label specified in the annotations is used to override the property's default label.

To explicitly define which field groups are to be displayed in the Adapt Filters popup, use the UI.FilterFacets annotation.
