Process Flow

Introduction

This control displays documents in their flow. It consists of process flow nodes (sap.suite.ui.commons.ProcessFlowNode) that represent the documents displayed, process flow connections (sap.suite.ui.commons.ProcessFlowConnection) that connect the nodes in the flow and process flow lanes (sap.suite.ui.commons.ProcessFlowLane) that represent the type of documents and provide an overview on document count and status.
