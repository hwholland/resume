Visual Business GeoMap Container Interaction Sample

  • Testing the Visual Business GeoMap Control with Container VOs hosting charts.
  • Option to remove a chart by changing model data (control needs to keep state).
  • Option to remove a chart by replacing the model (control needs to keep state).
  • Option to remove all charts by removing the VO aggregation (control needs to keep state).
  • Option to add charts again by adding a VO aggregation (control needs to keep state).
  • Option to open a Detail window for a container on click event.
  • Option to open a context menu for a container on right click event.

Remarks: Model changes can only be undone by reloading the page. Removing and Adding VOs should be repeatable!
