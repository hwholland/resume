new sap.landvisz.LandscapeViewer(sId?, mSettings?)

Constructor for a new LandscapeViewer.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers.

If the name of a setting is ambiguous (e.g. a property has the same name as an event), then the framework assumes property, aggregation, association, event in that order. To override this automatic resolution, one of the prefixes "aggregation:", "association:" or "event:" can be added to the name of the setting (such a prefixed name must be enclosed in single or double quotes).

The supported settings are:

  • Properties

    • defaultDependencyView : sap.landvisz.DependencyType (default: sap.landvisz.DependencyType.NETWORK_VIEW)
    • height : int
    • width : int
    • title : string
    • networkDependencyLabel : string (default: 'Network View')
    • boxDependencyLabel : string (default: 'Box View')
    • boxDependencyLevels : string[]
    • showDependencyNavigator : boolean (default: true)
    • visibleDependency : sap.landvisz.DependencyVisibility (default: sap.landvisz.DependencyVisibility.BOTH)
    • viewType : sap.landvisz.ViewType
    • navigationPath : string
    • hasParent : boolean (default: true)
    • selectionViewPosition : sap.landvisz.SelectionViewPosition (default: sap.landvisz.SelectionViewPosition.CENTER)
    • showClose : boolean (default: true)
    • componentViewLabel : string (default: 'Component View')
    • deploymentViewLabel : string (default: 'Deployment View')
    • solutionOptionType : sap.landvisz.OptionType
    • solutionType : sap.landvisz.SolutionType
    • showDeploymentTypeSection : boolean (default: false)
    • deploymentOptionsLabel : string (default: 'Deployment Options :')
    • deploymentOptionsTooltip : string
    • componentViewTooltip : string
    • deploymentViewTooltip : string
    • closeButtonTooltip : string
    • plugContentHeight : int (default: 0)
    • connectionLine : sap.landvisz.ConnectionLine (default: sap.landvisz.ConnectionLine.Line)

  • Aggregations

    • systems : sap.landvisz.LandscapeEntity[]
    • connectionEntities : sap.landvisz.ConnectionEntity[]
    • connectors : sap.landvisz.Connector[]
    • solutionOptions : sap.landvisz.Option[]
    • plugContent : sap.ui.core.Control

  • Associations

    • Events

      • close : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
      • loadSolutionView : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

    In addition, all settings applicable to the base type sap.landvisz.Option can be used as well.

    Parameters:
    stringsId?id for the new control, generated automatically if no id is given
    objectmSettings?initial settings for the new control
