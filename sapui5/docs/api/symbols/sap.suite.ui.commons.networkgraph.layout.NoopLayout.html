Class sap.suite.ui.commons.networkgraph.layout.NoopLayoutModule: sap/suite/ui/commons/networkgraph/layout/NoopLayout

extends LayoutAlgorithm
This is a simple layout algorithm that expects the positions of nodes to be already present. It only creates line coordinates (see sap.suite.ui.commons.networkgraph.Line#setCoordinates).
