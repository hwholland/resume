Namespace sap.ui.vtm.MatrixUtilitiesModule: sap/ui/vtm/MatrixUtilities

A set of utility functions for working with transformation matrices in a ISO 10303-42 format (in the form of a 1 dimensional array of 13 numbers)
Experimental API:
Since 1.50.0 This class is experimental and might be modified or removed in future versions.
