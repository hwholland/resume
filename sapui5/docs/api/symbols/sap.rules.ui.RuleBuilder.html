new sap.rules.ui.RuleBuilder(sId?, mSettings?)

Constructor for a new RuleBuilder control.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • types : sap.rules.ui.RuleType[] (default: DecisionTable,TextRule)
    • bindingContextPath : string
    • editable : boolean (default: true)

  • Aggregations

    • decisionTableConfiguration : sap.rules.ui.DecisionTableConfiguration
    • textRuleConfiguration : sap.rules.ui.TextRuleConfiguration

  • Associations

    • expressionLanguage : (sap.ui.core.ID | sap.rules.ui.services.ExpressionLanguage)

In addition, all settings applicable to the base type Control can be used as well.

Parameters:
stringsId?ID for the new control, generated automatically if no ID is given.
objectmSettings?Initial settings for the new control.
