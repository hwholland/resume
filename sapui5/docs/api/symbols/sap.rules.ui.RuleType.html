Enum sap.rules.ui.RuleTypeModule: sap/rules/ui/library

An enumeration that defines whether the rule is formulated as a table with multiple rules instead of a rule with a single associated condition.
Field Summary
sap.rules.ui.RuleType.DecisionTableSpecifies that the rule is formulated as a table that allows complex rules to be visualized according to an if-then-else logic.
sap.rules.ui.RuleType.TextRuleSpecifies that the rule is formulated as a single condition, which is written directly in a business language.
