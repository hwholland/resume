Constructor Detail
new sap.viz.ui5.controls.VizSlider(sId?, mSettings?)

Constructor for a new ui5/controls/VizSider.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • vizType : string (default: column)
    • uiConfig : object
    • width : sap.ui.core.CSSSize (default: 800px)
    • height : sap.ui.core.CSSSize (default: 200px)
    • valueAxisVisible : boolean (default: true)
    • showPercentageLabel : boolean (default: true)
    • showStartEndLabel : boolean (default: true)

  • Aggregations

    • dataset : sap.viz.ui5.data.Dataset
    • feeds : sap.viz.ui5.controls.common.feeds.FeedItem[]

  • Events

    • rangeChanged : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.viz.core.Control can be used as well.

Parameters:
stringsId?id for the new control, generated automatically if no id is given
objectmSettings?initial settings for the new control
