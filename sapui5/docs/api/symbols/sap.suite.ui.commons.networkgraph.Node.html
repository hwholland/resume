Constructor Detail
new sap.suite.ui.commons.networkgraph.Node()

Constructor for a new Node.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • collapsed : boolean (default: false)
    • selected : boolean (default: false)
    • group : string
    • key : string
    • shape : sap.suite.ui.commons.networkgraph.NodeShape (default: Circle)
    • icon : string
    • width : int (default: undefined)
    • height : int (default: undefined)
    • maxWidth : int (default: undefined)
    • x : float (default: 0)
    • y : float (default: 0)
    • showExpandButton : boolean (default: true)
    • showActionLinksButton : boolean (default: true)
    • showDetailButton : boolean (default: true)
    • titleLineSize : int (default: 1)
    • descriptionLineSize : int (default: -1)
    • iconSize : int (default: undefined)

  • Aggregations

    • actionLinks : sap.ui.core.Control[]
    • actionButtons : sap.suite.ui.commons.networkgraph.ActionButton[]
    • image : sap.suite.ui.commons.networkgraph.NodeImage

  • Events

    • press : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • hover : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • collapseExpand : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.suite.ui.commons.networkgraph.ElementBase can be used as well.

Gets content of aggregation actionButtons.

A list of custom action buttons. These buttons are displayed in the button area for each node. A node may have up to 4 buttons. The default 3 buttons (collapse/expand, details, and links) have priority over any other custom buttons that you add.
