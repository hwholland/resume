Class sap.ui.vk.ViewerModule: sap/ui/vk/Viewer

extends Control
Provides simple 3D visualization capability by connecting, configuring and presenting the essential Visualization Toolkit controls a single composite control.
Experimental API:
Since 1.32.0 This class is experimental and might be modified or removed in future versions.

new sap.ui.vk.Viewer(sId?, mSettings?)

Constructor for a new Viewer.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • enableOverlay : boolean (default: false)
    • enableSceneTree : boolean (default: true)
    • showSceneTree : boolean (default: true)
    • enableStepNavigation : boolean (default: true)
    • enableNotifications : boolean (default: true)
    • showStepNavigation : boolean (default: false)
    • showStepNavigationThumbnails : boolean (default: true)
    • enableToolbar : boolean (default: true)
    • enableProgressIndicator : boolean (default: true)
    • width : sap.ui.core.CSSSize
    • height : sap.ui.core.CSSSize
    • toolbarTitle : string (default: )
    • shouldTrackVisibilityChanges : boolean (default: false)
    • runtimeSettings : object (default: [object Object])
    • webGLContextAttributes : object (default: { antialias: true, alpha: true, premultipliedAlpha: false })
    • showAllHotspots : boolean (default: false)
    • hotspotColorABGR : int (default: 3221225727)
    • hotspotColor : sap.ui.core.CSSColor (default: rgba(255, 0, 0, 0.7529411764705882))

  • Aggregations

    • contentResources : sap.ui.vk.ContentResource[] (default)
    • overlay : sap.ui.vk.Overlay

  • Events

    • contentResourceChangesProcessed : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • sceneLoadingSucceeded : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • sceneLoadingFailed : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • sceneDestroying : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • selectionChanged : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • fullScreen : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • urlClicked : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • nodeClicked : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
stringsId?ID for the new Viewer control, generated automatically if no ID is given
objectmSettings?initial settings for the new Viewer control
