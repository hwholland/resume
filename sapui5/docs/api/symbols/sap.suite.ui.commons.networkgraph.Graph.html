Class sap.suite.ui.commons.networkgraph.GraphModule: sap/suite/ui/commons/networkgraph/Graph

extends SvgBase
The network graph control allows you to display data as a network of nodes connected by lines. The nodes can be circular or rectangular and can be joined into groups. You can define custom attributes for nodes and groups of nodes as well as apply layout algorithms that define the graph's appearance.
Since: 1.50.

new sap.suite.ui.commons.networkgraph.Graph()

Constructor for a new Graph.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • height : sap.ui.core.CSSSize (default: 100%)
    • width : sap.ui.core.CSSSize (default: 100%)
    • orientation : sap.suite.ui.commons.networkgraph.Orientation (default: LeftRight)
    • enableWheelZoom : boolean (default: true)
    • backgroundImage : sap.ui.core.URI

  • Aggregations

    • lines : sap.suite.ui.commons.networkgraph.Line[]
    • nodes : sap.suite.ui.commons.networkgraph.Node[]
    • groups : sap.suite.ui.commons.networkgraph.Group[]
    • legend : sap.ui.core.Control
    • layoutAlgorithm : sap.suite.ui.commons.networkgraph.layout.LayoutAlgorithm
    • statuses : sap.suite.ui.commons.networkgraph.Status[]

  • Associations

    • ariaDescribedBy : (sap.ui.core.ID | sap.ui.core.Control)[]
    • ariaLabelledBy : (sap.ui.core.ID | sap.ui.core.Control)[]

  • Events

    • graphReady : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • afterLayouting : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • beforeLayouting : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • zoomChanged : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • failure : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • selectionChange : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • searchSuggest : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • search : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.suite.ui.commons.networkgraph.SvgBase can be used as well.
