sap.gantt.GenericArray

A hybrid data type that can represent an array of string, or array of object. The result value parsed by this data type are "string[]" or "object[]"

Examples of valid values in js: 1. ["order", "activity"] 2. [{name:"order", idName:"OrderNo"},{name:"activity"}] 3. [{name:"order", idName:"OrderNo"},"activity"]

Examples of valid values in xml view: 1. "order,activity" 2. "order, activity" 3. "[order,activity]" 4. "[order, activity]" 5. '[{"name":"order", "idName":"OrderNo"},{"name":"activity"}]' 6. "[{'name':'order', 'idName':'OrderNo'},{'name':'activity'}]"

sap.gantt.ValueSVGPaintServer

A string type that represents SVG fill color values.

Allowed values are sap.ui.core.CSSColor and sap.m.ValueColor and LESS parameter

The empty string and invalid less parameter fall back to default black color.
