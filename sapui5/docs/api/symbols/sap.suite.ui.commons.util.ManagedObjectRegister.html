Class sap.suite.ui.commons.util.ManagedObjectRegisterModule: sap/suite/ui/commons/util/ManagedObjectRegister

extends Object
A registry which keeps a set of managed objects and allows you to lazy load them and destroy them at once. A lot of controls keep those object as private properties and destroies them one by one. Instead you can just use: 

this._objectRegister = new ManagedObjectRegister();
this._objectRegister.register("Button", function() {
 return new Button(...);
});

this._objectRegister.getButton()...

this._objectRegister.destroyAll();
