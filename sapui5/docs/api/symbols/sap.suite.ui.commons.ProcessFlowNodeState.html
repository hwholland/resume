Enum sap.suite.ui.commons.ProcessFlowNodeStateModule: sap/suite/ui/commons/library

Describes the state connected to the content it is representing in the Process Flow Node. The state is also displayed in the Process Flow Lane Header as a color segment of the donut.
Field Summary
sap.suite.ui.commons.ProcessFlowNodeState.CriticalCritical status for a created document: with this document, a critical issue occurred, for example, the business process can be interrupted.
sap.suite.ui.commons.ProcessFlowNodeState.NegativeNegative status for a created document: with this document, an issue occurred.
sap.suite.ui.commons.ProcessFlowNodeState.NeutralNeutral status for a created document: the document is in progress.
sap.suite.ui.commons.ProcessFlowNodeState.PlannedPlanned status for a document: the document is planned to be started.
sap.suite.ui.commons.ProcessFlowNodeState.PlannedNegativePlanned, but negative status for a document: the planned document has an issue but has not yet been started.
sap.suite.ui.commons.ProcessFlowNodeState.PositivePositive status for a created document: the document is done, finished, solved.
