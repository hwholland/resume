Namespace sap.ui.vtm.extensionsModule: sap/ui/vtm/library

Namespace for extensions that are provided with the sap.ui.vtm library.
Experimental API:
Since 1.50.0 This library is experimental and might be modified or removed in future versions.
Namespaces & Classes
DisplayStateCalculationExtensionAdds a behavior that calculates display state (visibility, opacity, highlight color) for the scene nodes in a viewport using information in the tree items, context display groups and override display groups.
InitialViewExtensionAdds a behavior that zooms to all (or a specific view) after the first viewable has loaded successfully.
LoadProgressExtensionAdds a behavior that shows a progress dialog when downloading/loading of viewables is occurring.
MessageStatusCalculationExtensionAdds a behavior that calculates the properties of the message status icons.
MessageStatusIconClickExtensionAdds a behavior that shows a sap.ui.vtm.MessagesPopover when a message status icon is clicked.
SceneNodeHoverHighlightExtensionAdds a behaviour which highlights a scene node when the mouse hovers over it.
SceneNodeHoverTooltipExtensionAdds a behaviour in which a tooltip (which can be specified by application code using a callback function) is displayed when the mouse is hovering over a scene node.
SelectionKeepingExtensionWorkaround for sap.ui.table.TreeTable behaviour in which selections are lost after collapse/expand.
SelectionLinkingExtensionAdds a behavior that links items the visibility of tree items in a sap.ui.vtm.Tree with the visibility of their associated scene nodes in the sap.ui.vtm.Viewport in the same sap.ui.vtm.Panel.
SyncSelectionOnTreeChangeExtensionWorkaround for sap.ui.table.TreeTable behaviour in which the selection can change when the model is updated, but no selection changed event is fired.
ViewLinkingExtensionAdds a behavior that links the camera views for all the sap.ui.vtm.Viewport instances associated with a sap.ui.vtm.Vtm instance.
ViewportSelectionLinkingExtensionAdds a behavior that links the selection state of tree items in a sap.ui.vtm.Tree with their corresponding scene nodes in the sap.ui.vtm.Viewport in the same sap.ui.vtm.Panel.
VisibilityIconClickExtensionAdds a behavior that updates visibility in the sap.ui.vtm.Tree when a visibility checkbox (eye) is clicked.
