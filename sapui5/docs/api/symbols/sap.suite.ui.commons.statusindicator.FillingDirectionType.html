Enum sap.suite.ui.commons.statusindicator.FillingDirectionTypeModule: sap/suite/ui/commons/library

The direction of animation.

The direction types Up, Down, Left, and Right are available when sap.suite.ui.commons.statusindicator.FillingType is set to Linear.
The direction types Clockwise and Counterclockwise are available when sap.suite.ui.commons.statusindicator.FillingType is set to Circular.
