Final Namespace sap.ui.vtm.MatrixModule: sap/ui/vtm/library

A float[] type representing a transformation matrix in a ISO 10303-42 format (in a 1 dimensional array of 13 numbers).

The sap.ui.vtm.MatrixComponent enumeration enumerates the array indices of the matrix components for this type
