new sap.suite.ui.microchart.HarveyBallMicroChart(sId?, mSettings?)

The configuration of the graphic element on the chart.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • total : float
    • totalLabel : string
    • totalScale : string
    • formattedLabel : boolean (default: false)
    • showTotal : boolean (default: true)
    • showFractions : boolean (default: true)
    • size : sap.m.Size (default: Auto)
    • colorPalette : string[] (default: [])
    • width : sap.ui.core.CSSSize
    • isResponsive : boolean (default: false)

  • Aggregations

    • items : sap.suite.ui.microchart.HarveyBallMicroChartItem[] (default)

  • Events

    • press : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
stringsId?ID for the new control, generated automatically if no ID is given
