Class sap.m.NotificationListItemModule: sap/m/NotificationListItem

extends NotificationListBase

The NotificationListItem control shows notifications to the user.

Structure

The notification item holds properties for the following elements:

  • description - additional detail text.
  • hideShowMoreButton - visibility of the "Show More" button.
  • truncate - determines if title and description are truncated to the first two lines (usually needed on mobile devices).

For each item you can set some additional status information about the item processing by adding a sap.m.MessageStrip to the processingMessage aggregation.

Constructor for a new NotificationListItem.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • description : string (default: )
    • truncate : boolean (default: true)
    • hideShowMoreButton : boolean (default: false)

  • Aggregations

    • processingMessage : sap.m.MessageStrip

In addition, all settings applicable to the base type sap.m.NotificationListBase can be used as well.
