Enum sap.gantt.config.BirdEyeRangeModule: sap/gantt/library

Define the range of data that bird eye would use to calculate visibleHorizon
Field Summary
sap.gantt.config.BirdEyeRange.AllRowsBird eye will calculate visibleHorizon based on all rows of current view, but this mode may cause low performance.

Field Detail
sap.gantt.config.BirdEyeRange.AllRows
Bird eye will calculate visibleHorizon based on all rows of current view, but this mode may cause low performance. Using this bird eye range, the result is not influenced by vertical scroll bar position as the total rows never changes.
sap.gantt.config.BirdEyeRange.VisibleRows
Bird eye will calculate visibleHorizon based on row data only in visible rows. Using this bird eye range, the result can be influenced by vertical scroll bar position as visible rows may change with the movement of the vertical scroll bar position.
