Class sap.m.PDFViewerModule: sap/m/PDFViewer

extends Control

This control enables you to display PDF documents within your app. It can be embedded in your user interface layout, or you can set it to open in a popup dialog.

Please note that the PDF Viewer control can be fully displayed on desktop devices only. On mobile devices, only the toolbar with a download button is visible.

new sap.m.PDFViewer(sId?, mSettings?)

Definition of PDFViewer control

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties

    • height : sap.ui.core.CSSSize (default: 100%)
    • width : sap.ui.core.CSSSize (default: 100%)
    • source : sap.ui.core.URI
    • errorMessage : string
    • errorPlaceholderMessage : string
    • popupHeaderTitle : string
    • title : string
    • showDownloadButton : boolean (default: true)

  • Aggregations

    • errorPlaceholder : sap.ui.core.Control
    • popupButtons : sap.m.Button[]

  • Events

    • loaded : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • error : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]
    • sourceValidationFailed : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
stringsId?id for the new control, generated automatically if no id is given
objectmSettings?initial settings for the new control
