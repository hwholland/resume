Enum sap.rules.ui.ExpressionTypeModule: sap/rules/ui/library

An enumeration that defines the different business data types for an expression
Field Summary
sap.rules.ui.ExpressionType.AllSpecifies that the expression can be of any of the supported business data types.
sap.rules.ui.ExpressionType.BooleanSpecifies that the expression must represent a Boolean data type: true, false.
sap.rules.ui.ExpressionType.DateSpecifies that the expression must represent a date only.
sap.rules.ui.ExpressionType.NumberSpecifies that the expression must represent a real number with or without dot-decimal notation.
sap.rules.ui.ExpressionType.StringSpecifies that the expression must represent a single-quoted UTF-8 encoded string.
sap.rules.ui.ExpressionType.TimeSpecifies that the expression must represent a time only.
sap.rules.ui.ExpressionType.TimeSpanSpecifies that the expression must represent a time difference in milliseconds.
sap.rules.ui.ExpressionType.TimestampSpecifies that the expression must represent a date and timestamp.
