Example for a Properties File

Bear in mind to replace the placeholder <GUID> in the first line with a real generated 32-character GUID. 

#!properties
# SAPUI5 TRANSLATION-KEY <GUID>
#XMSG
HELLO_WORLD = Hello World
#XBUT,10
BUTTON_SAVE = Save
#XFLD,30: Greetings displayed in the upper right corner of the screen
welcome = Welcome {0}
