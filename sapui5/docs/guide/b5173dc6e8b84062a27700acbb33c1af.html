Editing Status in Header

Draft handling and locking is available for apps created with smart templates. The editing status for an item can be viewed by clicking on the Status icon in the header next to the title. Additional draft administrative data is also accessed, such as the first and last name of the user who last changed the item. Note that if the user's full name is not available in the master data in the back-end system, only the technical user name displays.

Users also receive a system prompt when another user accesses unsaved changes.

For more information about editing statuses, see Editing Status.
