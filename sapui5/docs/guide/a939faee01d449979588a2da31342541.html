Which Smart Templates Are Available?

The following smart templates are available:

  • The SAP Web IDE contains the smart template tile Smart Template Application. When you choose this wizard, the generated app consists of a combination of the List Report and Object Page floorplans.

    Information about the list report and object page views and how to use smart templates is available in the following sections:

    • List Report View
    • Object Page View
    • How To Use Smart Templates

    For more information about SAP Fiori floorplans, see the SAP Fiori Design Guidelines.

  • Using the overview page wizard in SAP Web IDE, you can create overview pages and add cards to the page.

    For more information, go to Overview Pages: Create Interactive Overviews of a Subject Area.
