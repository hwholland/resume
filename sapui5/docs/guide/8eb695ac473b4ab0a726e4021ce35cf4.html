Disabling the Editing Status Filter

The editing status filter is enabled by default in the SAP Fiori element for the list report.

If required, you can disable this filter for your app. To do so, in the metadata.xml of the OData service, set sap:filterable to false for the navigation property of the DraftAdministrativeData as shown below:
<NavigationProperty xmlns:sap="http://www.sap.com/Protocols/SAPData" Name="DraftAdministrativeData" Relationship="draftinfrastructure.SalesOrderToDraftAdministrativeDataType
"FromRole="FromRole_SalesOrderToDraftAdministrativeDataType" ToRole="ToRole_SalesOrderToDraftAdministrativeDataType" sap:filterable="false"/>
