Enabling the Search Function

To enable the search function, sap:searchable must be set to true for the root entity set. To enable additional search fields, specify additional PropertyPath values as shown in the code sample below.

List Report: Search

Code Sample
Metadata XML
<EntitySet Name="SEPMRA_C_PD_Product"EntityType="SEPMRA_PROD_MAN.SEPMRA_C_PD_ProductType" sap:searchable="true" sap:content-version="1"/>
Annotation XML
<Annotations Target="SEPMRA_PROD_MAN.SEPMRA_C_PD_ProductType">
	...
	<Annotation Term="UI.SelectionFields">
		<Collection>
			<PropertyPath>Price</PropertyPath>
			<PropertyPath>Supplier</PropertyPath>
		</Collection>
	</Annotation>
