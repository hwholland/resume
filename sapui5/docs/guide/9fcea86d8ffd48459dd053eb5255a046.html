Hiding a Facet on the UI

You can hide or show reference facets on the UI using the See more or See less links.

The See more link appears in the bottom right corner when hidden facets available.

See Less
To enable hiding facets on the UI, you set the PartOfPreview annotation of the relevant reference facet to false as in the following example:
Sample Code 
<Record Type="UI.ReferenceFacet">			
<PropertyValue Property="Label" String="{@i18n>@ProductDescription}"/>
<PropertyValue Property="Target" AnnotationPath="to_ProductTextInCurrentLang/@UI.FieldGroup#PlainText"/>
		<Annotation Term="UI.PartOfPreview" Bool="false"/>	
	</Record>
Note This feature is available for reference facets that are implemented under a collection facet.
