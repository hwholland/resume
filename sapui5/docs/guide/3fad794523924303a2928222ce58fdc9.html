Further Post-Generation Steps

To enable or modify certain default features for apps based on SAP Fiori elements, you can modify the manifest.json file and adapt the annotations.

You have several options with which to manually enhance the generated app. They are described in the floorplan-specific configuration sections:
  • List Report and Object Page
  • Worklist
  • Analytical List Page
  • Overview Pages
Note These enhancements are not modifications. They are overwritten if the app is regenerated, which happens when you restart the SAP Web IDE wizard for the same app. The wizard is only intended for one-time generation.
